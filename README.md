# planet.bluesabre.org

**This site has been discontinued.** It used to aggregate Xfce and Xubuntu release announcements and blog posts; it's now a single static landing page pointing visitors to the real upstream sites (seand.xyz, xfce.org, xubuntu.org), plus a redirect for old post URLs to that landing page.

Powered by [Jekyll](https://github.com/jekyll/jekyll). Deployed automatically by [GitHub Actions](.github/workflows/deploy.yml) on every push to `master` &mdash; no manual deploy step.

## Build and Serve Locally

Start a local preview server

    jekyll serve

Build the page into `build/`

    ./build.sh
