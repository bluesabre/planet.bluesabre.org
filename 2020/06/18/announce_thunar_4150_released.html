ANNOUNCE: thunar 4.15.0 released

18 Jun 2020

This is a development release of Thunar aiming the upcoming release of Xfce 4.16.

As you may have noticed, Thunar's versioning scheme changed to follow the format adopted by other Xfce core components. We hope this doesn't cause any problem for packagers (since 4.15.0 > 1.8.15) and simplifies how to reason about compatibility between components.

This release includes all fixes and improvements present in Thunar 1.8.x plus the changes listed below. Before going into details, I would like to thank the usual hard work of translators and the many contributions from Theo, Reuben, Cyrille and Yousuf, thank you all, we really appreciate your efforts.

Oh, with regards stability, it's pretty good, I've been using git master for while now, but due big changes carried by Alexander (in preparation for gtk4 and making the code easier to maintain) expect a couple of rough edges. We hope to get feedback (and bug reports) from early adopters as soon as possible in order to fix all regressions in time for Xfce 4.16 ;)
