ANNOUNCE: tumbler 0.2.9 released

06 Aug 2020

This is a bugfix release from the stable xfce-4.14 branch. This release addresses the issue described in Bug #28 and partially solved by the previous release 0.2.28. The issue now is completely solved by adding the correct orientation handling when fallback code for generating jpeg thumbnail takes place.
